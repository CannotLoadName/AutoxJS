# Overview
Launch Auto.js and AutoX.js scripts through Python in Termux.<br/>
Run "pip install AutoxJS" to install this package.<br/>
Then run "python3 -m autojs -h" to learn how to use this package.<br/>
For new versions of AutoX.js, running "python3 -m autojs -c intent_component org.autojs.autoxjs/.external.open.RunIntentActivity" to change the package name may be necessary.<br/>
